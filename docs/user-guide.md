# User Guide

## Owner/Admin

Use Settings to maintain pricing, tax, discounts, invoice numbering, users, GitHub sync details, and backups.

## Operations

Use Customers for profile updates and Scheduling for pause requests or delivery status changes.

## Kitchen/Packing

Use Packing / Kitchen before preparation and packing. Allergy warnings and menu conflicts require acknowledgement.

## Billing

Use Invoice Generator to select a customer and billing period. Delivered days are charged. Skipped, customer-pause, and cancelled-before-prep days are visible but excluded.
